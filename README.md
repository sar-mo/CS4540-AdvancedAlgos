# CS 4540: Advanced Algorithms / CS 6515: Graduate Algorithms

This repository contains information / resources about CS 4540: Advanced Algorithms, taught at Georgia Tech. This repo is based on the Spring 2024 section of this course. It was taught by Dana Randall, with the TA's being Sarthak Mohanty (Head TA), Aadit Trivedi, and Arvind Ravipati.

This semester was the first time this course was taught in a while! Thus, much of the material we had to create / develop from scratch.

If you're an instructor or a TA of an algorithms course, feel free to use these resources for your classes!
